Python randompack decisions

1. Package name is randompack; API uses Randompack.Rng() / Randompack.Rng(engine).
2. Randompack.engines() returns a dict mapping engine -> description.
3. No top-level draw helpers (only Rng methods plus engines).
4. Distribution method names follow R exactly (unif, normal, skew_normal, lognormal,
   exp, gamma, chi2, beta, t, f, gumbel, pareto, weibull, mvn, int, perm, sample, raw).
5. Use unif/normal for defaults; do not bind u01/norm separately.
6. No parameter-name aliases; match R/Julia naming as-is.
7. Continuous draws support Float64 and Float32; default is Float64.
8. All continuous methods accept out= buffers; out must be contiguous.
9. dtype= controls output type for returning draws; dtype and out together are invalid.
10. Integer draws support int32 and int64 only (default int64); type chosen by dtype or out.
11. raw(n) returns a Python bytes object.
12. seed(seed, spawn_key=...): seed must fit int32; spawn_key entries must be in
    [0, 2^32-1] and converted to uint32.
13. randomize(), duplicate(), serialize()/deserialize() supported; serialize returns
    bytes; deserialize accepts only bytes (no bytearray).
14. full_mantissa(True/False) uses positional boolean only.
15. State-setters accept integer vectors only (no derived types); range-checked like
    Julia and converted to uint64.
16. pcg64_set_state uses low 64 bits only (high bits zeroed).
17. MVN: Float64 only; output shape is n x d (C order), call C with transp='T';
    copy Sigma and mu if non-contiguous; no symmetry check in Python.
18. Error handling: type/range errors raise TypeError/ValueError; C failures raise
    RuntimeError with last_error; unknown engine uses "spelling error" message.
19. Binding approach is Cython, bundling C sources.
20. Directory structure: python/randompack/_core.pyx (single file), python/src/ for
    C sources, tests in python/tests/.
21. Tests first; tests modeled after Fortran; docstrings modeled after Julia.
22. Tests: full coverage on default engine; other engines get a smoke test
    (create + seed + 3 unif draws).
